# Python_based_Trip_planner_AI_assistant_chatbot

##Project Summary

Ghafoor ka Dosth is an AI-driven conversational chatbot built with Python and Streamlit that helps users plan personalized trips to Dubai. Acting as a virtual Dubai tourism expert, the chatbot provides guidance on attractions, food, hotels, events, and customized travel itineraries through natural language interaction.
This project demonstrates the practical use of Generative AI, prompt engineering, and web app deployment for real-world travel planning use cases.
